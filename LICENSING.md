# 许可证

## 各版本许可证
| 版本    | 许可证         |
|---------|---------------|
| 1.0     | 无许可证       |
| 1.1     | 无许可证       |
| 1.2     | 无许可证       |
| 2.0     | 无许可证       |
| 3.0     | TELEMT UL 1   |

### 许可证类型
- **无许可证** = ***保留所有权利***
- **TELEMT UL1** - 适用于 `telemt` 源代码的使用许可证，鼓励：
  - 合理使用
  - 贡献代码
  - 分发传播
  - 但禁止不提及原作者
